# vuepress-theme-hope template

Template for vuepress-theme-hope.

You can begin your vuepress-theme-hope trip by creating a repo from this tempalte.

- Install deps:

  Run `yarn install`

- Dev Env:

  Run `yarn run serve`

- Build production files:

  Run `yarn run build`
